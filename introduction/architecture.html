Architecture

Description

Above is the complete architecture for the Ebiz Airlines Bot. By leveraging the Salesforce App Cloud, we can now offer customers an amazing and seamless flight booking experience.

Additionally, because we are using Heroku Connect and Salesforce Restful API to synchronize data between Heroku and Salesforce, our internal business users can manage all the customers, flights, routes, airports, messages, etc.


  1. Salesforce

    Salesforce is the leading cloud platform as a service and CRM on the planet!

    For this example app, Salesforce is our business system to manage members, membership options, and allows for membership management

  2. Heroku Connect

    Using bi-directional synchronization between Salesforce and Heroku Postgres, Heroku Connect unifies the data in your Postgres database with members, membership contacts, memberships, and membership tiers

    NO coding or complex configuration is required

  3. Heroku Postgres - HEROKU ADDON

    The SQL database service for developers.

    Easy to Set Up, Secure by Default

  4. Wit.AI

    Wit is a natural language interface for applications capable of turning sentences into structured data.

    The API authenticates with OAuth2 and returns JSON formatted responses.

  5. NodeJS

    Using bi-directional synchronization between Salesforce and Heroku Postgres, Heroku Connect unifies the data in your Postgres database with customers, flights, routes, airports, messages, etc.

    NO coding or complex configuration is required
